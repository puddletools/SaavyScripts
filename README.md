# SaavyScripts
Series of scripts written for SavvyCAN for Fisker vehicles.
Use at your own risk. For educational purposes only.

- GetFOBs.js: Get number of FOBs synced with vehicle
- GetNFC.js: Get number of NFC cards synced with vehicle
- GetVin.js: Get vehicle VIN
- GetPKCHW.js: Get PKC Hardware Number
- GetPKCPart.js: Get PKC part number
- GetPKCSW.js: Get PKC SW number
- GetPKCSWver.js: Get PKC SW version
